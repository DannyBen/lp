Lovely print
==================================================

---

The tinyest gem possible. Print anything as YAML with `lp your_object`.

---

Installation
--------------------------------------------------

    $ gem install lp



Usage
--------------------------------------------------

```ruby
require 'lp'

lp anything
```

--------------------------------------------------

- `p` is messy
- `pp` is messy
